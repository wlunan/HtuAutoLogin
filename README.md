## 河南师范大学校园网自动登录脚本

**版本：2.0 | 更新日期：2025-12-18 | 作者：lunan**

### 🚀 核心功能

专为河南师范大学师生打造的**一键智能登录**工具，自动保存账号密码，实现校园网秒级登录。

### ✨ 主要特色

* 仅在校园网域名生效 (`10.101.2.194`)
* 不收集任何数据，所有信息本地存储
* 开源代码，可自由审查

### 📋 使用场景

学校公共区域校园网登陆

### 🛠️ 安装使用

1. **安装Tampermonkey**等浏览器扩展
2. **点击安装脚本**（链接或粘贴代码）
3. **首次使用**：输入账号密码并勾选"记住凭证"
4. **后续登录**：校园网页面加载后自动登录

### ⚠️ 安全提示

- 请勿在公共设备上使用自动保存功能
- 定期清理浏览器缓存和脚本数据
- 如怀疑信息泄露，立即使用右下角"清除凭证"按钮
